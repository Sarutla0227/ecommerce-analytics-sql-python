# E-Commerce Sales & Logistics Performance Dashboard

## Project Overview
This project analyzes an e-commerce dataset to evaluate sales performance and delivery efficiency. 
Using PostgreSQL and Power BI, I built a relational data model, calculated key business KPIs, and designed an interactive dashboard for performance monitoring.

---

## Objectives
- Analyze revenue trends over time
- Calculate Average Order Value (AOV)
- Measure Late Delivery Rate
- Identify top-performing product categories

---

## Tools Used
- PostgreSQL (Database & SQL queries)
- SQL (Joins, Aggregations, KPI calculations)
- Power BI (Data modeling, DAX, Dashboard)
- GitHub (Version control & documentation)

---

## Key Results
- Total Orders: ~99K
- Total Revenue: ~$15.84M
- Average Order Value (AOV): ~$159
- Late Delivery Rate: ~8.11%
- Top Revenue Category: Health & Beauty

---

## Dashboard Features
- KPI cards (Revenue, Orders, AOV, Late Delivery Rate)
- Monthly revenue trend analysis
- Late delivery rate trend
- Top 10 product categories by revenue
- Interactive filters (Order Status, Date Range)

---

## Business Insights
- Revenue peaked mid-year before experiencing seasonal decline.
- AOV remains stable, indicating consistent customer spending.
- 8.11% late deliveries suggest logistics optimization opportunity.
- Health & Beauty category contributes highest revenue.
